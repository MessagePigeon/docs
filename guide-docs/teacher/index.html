教师

消息​

  • 发送消息仅可选择在线设备
  • 发送消息默认自带提示音，请确保学生端设备声音开启
  • 教师姓名会在学生端消息上显示为 由XXX老师发送的消息，建议使用真实姓名，也可添加学科
  • 历史记录 页面可以撤回消息
info

语音播报由于技术原因暂未上线，将尽快完善

设备绑定​

  • 设备绑定可由教师前往教室手动添加（输入连接代码或扫描二维码），也可联系管理员后台添加
  • 手动添加后需在教室设备上同意绑定才算绑定成功

账号安全​

  • 如果您的密码是由管理员后台生成的，建议更换密码后重新登录
  • 如果忘记密码，可联系管理员重置为随机密码
