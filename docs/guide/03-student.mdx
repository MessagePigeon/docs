:::caution
请不要**重复打开**软件，_由于技术原因暂未实现重复打开提示_。如遇打开软件后**没有反应**，可能是因为**软件已经运行**，可以在**系统托盘**查找**小图标**点击显示
:::

## 开机自启动

1. 打开**运行** <kbd>Windows</kbd>+<kbd>R</kbd>
2. 输入 `shell:startup` 后确定，会打开**开机自启动文件夹**
3. 将桌面上飞鸽传书的**快捷方式**复制到文件夹中

## 登录

- **服务器地址**和**Key**请联系管理员获取，不能在**多台**设备上登录同一 Key，会自动退出**上一台**登录的设备
- `http` 和 `ws` 协议请按照管理员要求设置

## 消息

- 消息有**提示音**，请打开设备**声音**
- 如果教师发送消息时设置**关闭延迟时间**，将在消息窗口标题上显示**倒计时**，这段时间内**无法关闭和最小化**
- 消息的开启和关闭**状态**会在教师端显示
- 如果意外关闭消息，可在历史记录内**重新显示**，但是教师端消息**状态**并不会重新显示开启
