## 消息

1. 发送消息仅可选择**在线**设备
2. 发送消息默认自带**提示音**，请确保学生端设备声音开启
3. **教师姓名**会在学生设备上显示为 `由XXX老师发送的消息`，建议使用**真实姓名**，也可添加**学科**
4. 如需**撤回**，可至 `历史记录` 页面。消息如果仍在设备上显示，可点击**关闭**按钮

:::info
语音播报由于技术原因暂未上线，将尽快完善
:::

## 设备绑定

1. 设备绑定可由教师前往教室**手动添加**（输入连接代码或扫描二维码），也可联系管理员**后台添加**
2. **手动添加**后需在教室设备上**同意绑定**才算绑定成功

## 账号安全

1. 如果您的密码是由管理员**后台生成**的，建议**更换密码**后**重新登录**
2. 如果**忘记密码**，可联系管理员重置为**随机**密码
