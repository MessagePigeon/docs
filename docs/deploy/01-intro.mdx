title: 部署简介
---

:::info
此文档为部署教程，如需参与开发，请查看项目仓库 README
:::

:::tip
如遇部署问题，欢迎在底部添加交流群咨询
:::

本文档仅提供**后端**和**前端**（包括**教师端**和**管理员端**）部署  
**学生端**请自行前往 [Github Release](https://github.com/MessagePigeon/client-student/releases) 下载

:::caution
学生端**大版本**必须和后端**大版本**相同，否则可能会出现**兼容性问题**  
例：后端版本为**1**.x.x，则学生端版本也须为**1**.x.x
:::
