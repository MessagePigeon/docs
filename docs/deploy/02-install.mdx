---

## 环境

请事先安装 `Node` 环境和 `pnpm` 包管理器

:::caution
必须使用 `pnpm`，其他包管理器如 `yarn` 或 `npm` 已被禁用
:::

## 下载

请在 Github Release 下载打包文件，下载地址：

- [后端](https://github.com/MessagePigeon/server/releases)
- [前端](https://github.com/MessagePigeon/client-teacher/releases)

## 安装依赖

解压后在项目文件夹内

```bash
pnpm i
```
